# Vulnerability Assessment Report

## Website Tested
scanme.nmap.org

## Scope
Read-only passive security assessment

## Tools Used
- Nmap
- SecurityHeaders
- Browser DevTools

## Findings
- Missing Content-Security-Policy
- Missing X-Frame-Options
- Missing X-Content-Type-Options
- Missing Referrer-Policy
- Missing Permissions-Policy
- HTTP without HTTPS redirection
- SSH exposed on Port 22

## Risk Levels
High, Medium, and Low findings were identified.

## Evidence
Screenshots and PDF report are included in this repository.
